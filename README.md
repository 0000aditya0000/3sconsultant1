# 3S Consultant — Financial Consulting Website

A premium, modern financial consulting website for **3S Consultant**, a Noida-based firm providing loans and insurance solutions.

## Tech Stack

- **React.js** — Component-based UI
- **React Router** — Multi-page navigation
- **Tailwind CSS v4** — Utility-first styling
- **Framer Motion** — Scroll animations, page transitions, counters
- **Lucide React** — Professional icons
- **Vite** — Fast development and build tooling

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Pages

| Page | Route | Description |
|------|-------|-------------|
| Home | `/` | Hero, why choose us, stats, services preview |
| About | `/about` | Company mission, vision, values |
| Loan Services | `/loans` | 10 loan products with details |
| Insurance | `/insurance` | Motor, life, health insurance |
| Process | `/process` | 5-step timeline + documents |
| FAQ | `/faq` | Accordion-style Q&A |
| Contact | `/contact` | Form, phone numbers, map |

## Project Structure

